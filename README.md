# All India Consumer Price Index (CPI) Analysis – Excel Project

## Project Overview

This project analyzes the All India Consumer Price Index (CPI) dataset using Microsoft Excel. The objective is to identify inflation trends, category-wise price movements, and monthly/yearly changes through interactive dashboards and data analysis.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting
* Excel Functions

## Dataset

The dataset contains monthly CPI values for various expenditure categories across India.

## Analysis Performed

* Overall CPI trend analysis
* Year-over-Year (YoY) inflation analysis
* Monthly inflation trend
* Category-wise CPI comparison
* Highest and lowest inflation periods
* Contribution of different expenditure categories

## Key Insights

* Identified years with the highest inflation.
* Compared inflation across different expenditure categories.
* Created interactive dashboards using slicers.
* Highlighted monthly and yearly CPI trends for better decision-making.
